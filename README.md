🚀 Simple CRUD Application (Laravel + AJAX + DataTables)

This is a simple CRUD (Create, Read, Update, Delete) web application built with Laravel, featuring dynamic data handling using AJAX and a responsive UI powered by Milligram CSS. The project also integrates DataTables for enhanced table functionality like searching, sorting, and pagination.

✨ Features
🔧 Built with Laravel Framework
⚡ AJAX-based CRUD operations (no page reload)
📊 DataTables integration for advanced table UI
🎨 Clean and minimal UI using Milligram CSS
🔍 Real-time data update and smooth user experience
📱 Responsive design for all devices
🛠️ Tech Stack
Laravel (Backend)
PHP
AJAX (jQuery)
DataTables
Milligram CSS
MySQL
📌 Project Highlights
Fast and interactive UI without page reload
Clean MVC architecture using Laravel
Easy to extend and customize for larger applications
Beginner-friendly CRUD structure for learning purposes

⚙️ Installation

git clone https://github.com/Wasti66/Laravel-CRUD-Application
cd Laravel-CRUD-Application
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
